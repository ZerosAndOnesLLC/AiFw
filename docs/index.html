Everything you need, nothing you don't

A complete firewall platform with modern defaults. No bolt-on packages, no legacy PHP.

🔥

Stateful Firewall

Full pf stateful filtering with scheduling, aliases, traffic shaping (CoDel, HFSC, PRIQ), and rate limiting. IPv4 + IPv6 out of the box.

🔐

WireGuard + IPsec

Native WireGuard tunnels with per-peer config generation. IPsec ESP/AH in tunnel or transport mode. Zero-config road-warrior VPN.

🚨

IDS/IPS Engine

Built-in Suricata-compatible engine with Sigma and YARA rule support. Real-time alerts, flow tracking, and per-rule suppressions.

🤖

AI Threat Detection

Behavioral detectors for port scans, DDoS, brute force, C2 beacons, and DNS tunneling. Auto-response with configurable TTL blocks.

🌐

NAT, DNS & DHCP

SNAT, DNAT, port forwarding, 1:1 NAT, NAT64 and NAT46. Built-in DNS resolver, DHCP server with HA failover and DDNS.

📊

Live Dashboard

WebSocket-powered real-time metrics. Connection tracking, bandwidth graphs, memory breakdown, and live NAT flow topology.

🔁

HA Clustering

CARP virtual IPs, pfsync state synchronization, and config sync between cluster nodes. Automatic failover detection.

🔑

Granular RBAC

34 granular permissions across every subsystem. TOTP 2FA, OAuth/SSO, API keys, and custom roles. Built-in Certificate Authority.

⚙️

Commit Confirm

Every config change can be staged and auto-reverted on timeout if not confirmed. Versioned config history with diff and restore.
